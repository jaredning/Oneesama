# Oneesama v2.0
## Experimental Android client for Dynasty Reader
### Current version: v0.2.1

This a fork of the original Oneesama app, which can be found here: [http://defiantcatgirl.github.io/Oneesama/](http://defiantcatgirl.github.io/Oneesama/)

The main improvements over the original app are as follows:
- Dependencies have been modernized so the code can compile again
- External storage access - you can save chapters to an SD card, rather than being limited to the internal storage of your device
- Fix the Browse > Series page not working, as well as add pagination to easily page through all series available on Dynasty Scans
- Separate download chapter button so that you can queue up multiple chapters for download without having to open each chapter in the app
- Numerous performance improvements to ensure that chapters stored to an SD card load nearly as quickly as chapters stored to internal storage, especially for older devices

As a heads up, this APK only supports **Android 6.0 and above** - the original app supports back to **Android 4.1**.

The app is optimized with Android 7.1.1 in mind, since this is the version of Android running on my personal tablet.
This fork exists entirely because my tablet supports an external SD card and only has 16 GB of internal storage.

You can download the latest APK [here](https://github.com/jaredning/Oneesama/releases/download/v0.2.1/Oneesama_v0.2.1.apk)
